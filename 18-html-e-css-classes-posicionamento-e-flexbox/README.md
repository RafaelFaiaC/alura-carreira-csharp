# HTML e CSS — Classes, Posicionamento e Flexbox

Projeto desenvolvido como parte da **Carreira de Desenvolvimento Back-End .NET** da [Alura](https://www.alura.com.br/).

Este repositório contém o código criado durante o curso  
➡️ **[HTML e CSS: classes, posicionamento e Flexbox](https://cursos.alura.com.br/course/html-css-classes-posicionamento-flexbox)**

---

## 📘 Sobre o Curso

O objetivo deste curso é aprofundar a construção de páginas web com **HTML e CSS**, explorando o uso de **classes**, **posicionamento de elementos** e **Flexbox** para organizar o layout de forma mais eficiente.

Durante o desenvolvimento, foi criada uma página de apresentação pessoal com estrutura semântica, destaque visual para trechos do texto, botões de navegação e disposição dos elementos lado a lado, utilizando Flexbox para compor a interface.

Também foram aplicados conceitos importantes como padronização de classes, importação de fontes externas e definição de estilos visuais para tornar a página mais organizada e agradável.

## 📚 Conteúdos Abordados

### 🧱 Estruturação e Organização do HTML
- Criação de seções com a tag `section`
- Uso de `div` para organização de blocos
- Estrutura semântica com `main`, `h1`, `p`, `a` e `img`
- Boas práticas de indentação e organização do código

### 🎨 Classes e Boas Práticas
- Utilização de classes para estilização com CSS
- Nomeação de classes seguindo padrões organizados
- Separação entre estrutura (HTML) e estilo (CSS)
- Hierarquia entre elementos pai e filho

### 📐 Posicionamento e Box Model
- Entendimento do **Box Model**:
  - `margin`
  - `border`
  - `padding`
- Controle de espaçamento interno e externo
- Uso de `box-sizing` para controle de layout
- Reset de estilos padrão do navegador

### 📏 Unidades e Medidas
- Uso de unidades relativas como **viewport (`vh`, `vw`)**
- Ajuste de largura e altura dos elementos
- Consulta de medidas diretamente no Figma

### 🔍 Inspeção e Debug
- Uso das **Developer Tools** para inspecionar elementos
- Testes de layout diretamente no navegador
- Ajustes visuais em tempo real

### 📐 Flexbox e Layout
- Uso de `display: flex` para organização de layout
- Alinhamento com:
  - `justify-content`
  - `align-items`
- Posicionamento em linha e em coluna
- Uso de `gap` para espaçamento entre elementos
- Distribuição equilibrada entre texto e imagem

### ✍️ Tipografia e Estilo
- Alteração de fontes com `font-family`
- Controle de tamanho com `font-size`
- Ajuste de peso da fonte
- Importação de fontes externas com **Google Fonts**

### 🎯 Estilização de Elementos
- Criação de botões estilizados com CSS
- Remoção de sublinhado com `text-decoration`
- Aplicação de `border-radius` para arredondamento
- Definição de cores para fundo e texto
