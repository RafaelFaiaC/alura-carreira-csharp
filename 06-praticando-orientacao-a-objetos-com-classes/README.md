# 🧱 Praticando C# — Orientação a Objetos com Classes

Projeto desenvolvido como parte da **Carreira de Desenvolvimento Back-End .NET** da [Alura](https://www.alura.com.br/).

Este repositório contém os exercícios desenvolvidos durante o curso:  
➡️ **[Praticando C#: orientação a objetos com classes](https://cursos.alura.com.br/course/praticando-csharp-orientacao-objetos-classes)**

---

## 📘 Sobre o Curso

O objetivo deste curso é **praticar os conceitos fundamentais de Orientação a Objetos em C#**, com foco na **criação e uso de classes**, atributos, métodos, encapsulamento e organização do código.

Os exercícios simulam cenários comuns do mundo real, reforçando a modelagem de entidades e regras de negócio por meio de **aplicações de console simples e independentes**.

---

## 🛠 Tecnologias Utilizadas

- **C# / .NET**
- **Console Application**
- **Visual Studio** (ambiente de desenvolvimento)

---

## 📂 Exercícios Desenvolvidos

Os desafios abaixo estão organizados em **projetos independentes**, cada um focado em um cenário específico para fixação dos conceitos de orientação a objetos.

---

### ▶️ **CadastroLivrosBiblioteca**
Cadastro simples de livros em uma biblioteca.

- Criação da classe `Livro`
- Uso de atributos para armazenar dados
- Exibição das informações no console

---

### ▶️ **EmissaoPassagemViagem**
Simulação de emissão de passagens de viagem.

- Modelagem de uma passagem com dados do passageiro
- Encapsulamento de informações
- Exibição dos detalhes da viagem

---

### ▶️ **ContaBancaria**
Implementação de uma conta bancária.

- Criação da classe `ContaBancaria`
- Operações de depósito e saque
- Controle de saldo via métodos

---

### ▶️ **GerenciamentoFuncionarios**
Gerenciamento básico de funcionários.

- Classe `Funcionario`
- Uso de métodos para exibição de dados
- Simulação de regras simples de negócio

---

### ▶️ **RetanguloCalculoArea**
Representação de um retângulo e cálculo de área.

- Classe `Retangulo`
- Cálculo de área a partir de largura e altura
- Separação de responsabilidades

---

### ▶️ **ValidacaoAcessoFilmes**
Validação de acesso a conteúdos por classificação indicativa.

- Classe para representar o usuário
- Verificação de idade mínima
- Retorno de permissões via console

---

### ▶️ **ControleEstoque**
Controle básico de produtos em estoque.

- Classe `Produto`
- Atualização de quantidade
- Exibição do estado atual do estoque

---

### ▶️ **PedidoCompra**
Modelagem de um pedido de compra.

- Classe representando o pedido
- Atributos para dados do cliente e valores
- Exibição de resumo do pedido

---

### ▶️ **AgendamentoMedico**
Simulação de um agendamento médico.

- Classe `Consulta`
- Armazenamento de dados do paciente
- Exibição das informações agendadas

---

### ▶️ **ProdutoDigital**
Representação de um produto digital com informações técnicas.

- Classe específica para produto digital
- Uso de propriedades para metadados
- Exibição das informações no console
