## 📘 Sobre o Curso

O objetivo deste curso é apresentar os fundamentos de **HTML** e **CSS**, começando pela configuração do ambiente de desenvolvimento e avançando até a construção da estrutura básica de uma página web.

Ao longo do curso, são abordados conceitos essenciais como a diferença entre um arquivo HTML e um documento em editores como Google Docs, a importância da documentação do **W3C**, a criação de estruturas semânticas com tags apropriadas, o uso correto do `<!DOCTYPE html>`, além da separação entre conteúdo e apresentação.

Também são explorados os primeiros passos com **CSS**, incluindo a criação de um arquivo externo para estilização, a aplicação de cores, o uso de paletas de referência e a integração entre HTML e CSS para compor uma página simples e organizada.

## 📚 Conteúdos Abordados

### 🧱 Estruturação da Página
- Criação da estrutura básica de um documento HTML
- Uso de tags semânticas como `main`, `h1`, `p`, `a` e `img`
- Diferença entre o conteúdo do `<head>` e do `<body>`
- Utilização correta de `<!DOCTYPE html>`

### 🌐 Fundamentos de HTML
- O que é HTML e por que ele é considerado uma linguagem de marcação
- Organização de elementos e tags em um documento
- Importância da documentação oficial da W3C
- Configuração de metadados com tags `meta`

### 🖼 Imagens e Acessibilidade
- Inserção de imagens na página
- Criação de texto alternativo com `alt`
- Organização visual de elementos com foco em clareza e acessibilidade

### 🧰 Ambiente de Desenvolvimento
- Instalação e uso do **Visual Studio Code**
- Criação de pastas e abertura do projeto no editor
- Uso da extensão **Live Server** para visualizar alterações em tempo real
- Acesso às **Developer Tools** do navegador

### 🎨 Introdução ao CSS
- O que é CSS e sua função na estilização da página
- Criação de arquivo externo para estilos
- Integração do CSS ao HTML com a tag `link`
- Aplicação de cores no fundo, no texto e em destaques

### 🎯 Cores e Identidade Visual
- Uso de cores hexadecimais no CSS
- Consulta de paletas de cores
- Extração de cores diretamente do layout no Figma
- Destaque de trechos específicos do texto com estilos próprios

## 📂 Estrutura do Projeto

O projeto é composto por poucos arquivos, organizados de forma simples:

- **`index.html`**  
  Contém a estrutura principal da página, os textos, links e a imagem.

- **`style.css`**  
  Responsável pela estilização visual da página.

- **`imagem.png`**  
  Imagem utilizada na composição do conteúdo principal.
