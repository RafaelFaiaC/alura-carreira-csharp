# 🔁 Praticando C# — Orientação a Objetos com Polimorfismo

Projeto desenvolvido como parte da **Carreira de Desenvolvimento Back-End .NET** da [Alura](https://www.alura.com.br/).

Este repositório contém os exercícios desenvolvidos durante o curso:  
➡️ **[Praticando C#: orientação a objetos com polimorfismo](https://cursos.alura.com.br/course/praticando-csharp-orientacao-objetos-polimorfismo)**

---

## 📘 Sobre o Curso

O objetivo deste curso é **praticar e consolidar o conceito de Polimorfismo**, utilizando herança, sobrescrita de métodos e comportamento dinâmico em tempo de execução.

Os exercícios demonstram como diferentes classes podem compartilhar uma mesma base, mas **responder de formas distintas a uma mesma ação**, tornando o código mais flexível, extensível e alinhado aos princípios da Orientação a Objetos.

---

## 🛠 Tecnologias Utilizadas

- **C# / .NET**
- **Console Application**
- **Visual Studio** (ambiente de desenvolvimento)

---

## 📂 Exercícios Desenvolvidos

Os desafios abaixo estão organizados em **projetos independentes**, cada um focado na aplicação prática do polimorfismo.

---

### ▶️ **CalculadoraSobrecargaMetodos**
Calculadora com diferentes formas de execução.
- Uso de **sobrecarga de métodos**.
- Mesma funcionalidade com múltiplas assinaturas.

---

### ▶️ **RelatorioFuncionarios**
Geração de relatórios para diferentes tipos de funcionários.
- Classe base `Funcionario`.
- Comportamento polimórfico para exibição e cálculo de dados.

---

### ▶️ **NotificacoesSistema**
Envio de notificações por diferentes canais.
- Implementações distintas para cada tipo de notificação.
- Uso de sobrescrita de métodos.

---

### ▶️ **ExecucaoTarefasAgendadas**
Simulação de tarefas executadas em horários distintos.
- Classe base para tarefas.
- Execução variando conforme o tipo da tarefa.

---

### ▶️ **ExibindoMidias**
Exibição de diferentes tipos de mídia.
- Conteúdos como imagem, vídeo e áudio.
- Método comum com comportamentos específicos.

---

### ▶️ **ReservasExperiencias**
Sistema de reservas para experiências variadas.
- Diferentes tipos de experiências.
- Regras específicas aplicadas via polimorfismo.

---

### ▶️ **AvaliacaoConteudosPlataforma**
Avaliação de conteúdos em uma plataforma.
- Tipos distintos de conteúdo avaliável.
- Lógica polimórfica para cálculo ou exibição da avaliação.

---

### ▶️ **SimuladorTransporteUrbano**
Simulação de meios de transporte urbano.
- Veículo como classe base.
- Comportamentos específicos para cada transporte.

---

### ▶️ **CalculoJurosPerfisEmprestimo**
Cálculo de juros conforme o perfil do empréstimo.
- Estratégias distintas de cálculo.
- Substituição de condicionais por polimorfismo.

---

### ▶️ **AcoesBotoesInterfaceGrafica**
Simulação de ações em botões de interface.
- Botão base com ação genérica.
- Implementações específicas para cada botão.
