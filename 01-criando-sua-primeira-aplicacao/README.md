# C# — Criando sua Primeira Aplicação
Projeto desenvolvido como parte da **Carreira de Desenvolvimento Back-End .NET** da [Alura](https://www.alura.com.br/).

Este repositório contém o código criado durante o curso  
➡️ **[C#: criando sua primeira aplicação](https://cursos.alura.com.br/course/csharp-criando-primeira-aplicacao)**

---

## 📘 Sobre o Curso
O objetivo do curso é introduzir os fundamentos da linguagem C# e da plataforma .NET por meio do desenvolvimento de uma aplicação de console simples, utilizando variáveis, coleções, condicionais e métodos.

---

## 🛠 Tecnologias Utilizadas
- **C# / .NET**
- **Visual Studio** (ambiente de desenvolvimento)
- Aplicação baseada em **Console**

---

## 📚 Conteúdos Abordados

### 🔧 Configuração de Ambiente
- Preparação do ambiente no Windows, Mac e Linux  
- Criação e configuração de um projeto console em C#

### 🧠 Fundamentos da Linguagem
- Conceito de linguagem **fortemente tipada**
- Declaração e uso de variáveis (`string`, `int`, etc.)
- Exibição de informações com `Console.WriteLine`

### 🔀 Controle de Fluxo
- Construção de um menu interativo no console  
- Leitura de entradas do usuário  
- Uso de estruturas condicionais (`if`, `else`, `switch`)

### 📦 Estruturas de Dados
- Manipulação de **listas** (`List<T>`)
- Inserção de elementos com `.Add`
- Iteração com `for` e `foreach`
- Criação e uso de **dicionários** (`Dictionary<TKey, TValue>`)
- Acesso a valores por meio de chaves únicas

### 🔁 Modularização
- Criação de **métodos** para separar responsabilidades
- Implementação de métodos com retorno

---

## 🧪 Funcionalidades da Aplicação Desenvolvida
A aplicação criada no curso permite:

- Registrar bandas
- Listar bandas registradas
- Atribuir notas a bandas
- Calcular e exibir a média de avaliações
- Navegar por um menu interativo no console
