# 🔁 Praticando C# — Strings e Expressões Regulares

Projeto desenvolvido como parte da **Carreira de Desenvolvimento Back-End .NET** da [Alura](https://www.alura.com.br/).

Este repositório contém os exercícios desenvolvidos durante o curso:  
➡️ **[Praticando C#: strings e expressões regulares](https://cursos.alura.com.br/course/praticando-csharp-strings-regex)**

---

## 📘 Sobre o Curso

O objetivo deste curso é **praticar e consolidar a manipulação de strings no C#**, além de explorar o uso de **expressões regulares (Regex)** para identificação, validação e transformação de padrões em textos.

Os exercícios abordam desde operações básicas com strings até cenários mais avançados, como extração de informações, validação de formatos e padronização de dados.

---

## 🛠 Tecnologias Utilizadas

- **C# / .NET**
- **Manipulação de Strings**
- **Regex (Expressões Regulares)**
- **Console Application**
- **Visual Studio** (ambiente de desenvolvimento)

---

## 📂 Exercícios Desenvolvidos

Os desafios abaixo estão organizados em **projetos independentes**, cada um focado na manipulação de texto e uso de expressões regulares.

---

### ▶️ **EncontrandoPalavrasChave**
Busca e validação de palavras dentro de um texto.
- Verificação da existência de termos específicos em uma string
- Comparação de texto com controle de maiúsculas e minúsculas
- Uso de métodos como `Contains` e `Equals`

---

### ▶️ **ContandoCaracteres**
Contagem detalhada de caracteres em uma string.
- Percorre a string caractere por caractere
- Contabiliza a quantidade total de caracteres
- Possível separação entre letras, números e espaços

---

### ▶️ **SubstituindoPalavrasFrase**
Substituição de termos dentro de frases.
- Alteração de palavras específicas usando `Replace`
- Geração de uma nova string com o conteúdo atualizado
- Demonstração da imutabilidade das strings

---

### ▶️ **DividindoString**
Separação de dados textuais em partes menores.
- Uso do método `Split` para quebrar uma string
- Transformação da string em uma coleção de palavras
- Iteração sobre os elementos resultantes

---

### ▶️ **PadronizandoAlertasMaiusculas**
Normalização de mensagens.
- Conversão de textos para letras maiúsculas com `ToUpper`
- Padronização de saídas para exibição consistente

---

### ▶️ **FormatandoRelatorio**
Formatação e alinhamento de textos.
- Uso de interpolação de strings (`$""`)
- Aplicação de `PadLeft` e `PadRight` para alinhamento
- Organização visual de dados em formato de relatório

---

### ▶️ **VerificandoStringsNumericas**
Validação segura de valores numéricos.
- Uso de `int.TryParse` para verificar se a string é numérica
- Evita exceções durante conversão
- Tratamento de entradas inválidas

---

### ▶️ **ExtraindoValorMoeda**
Extração e conversão de valores monetários.
- Remoção de símbolos como `R$` de uma string
- Conversão do valor para tipo numérico (`double` ou `decimal`)
- Tratamento de formatação de valores

---

### ▶️ **SubstituindoEspacosBranco**
Tratamento de espaços em strings.
- Remoção de espaços desnecessários com `Trim`
- Substituição de múltiplos espaços por valores padronizados
- Limpeza de dados textuais

---

### ▶️ **ValidandoDatas**
Validação de formatos de data com Regex.
- Uso de expressões regulares para identificar padrões de data
- Verificação de formatos como `dd/mm/aaaa`
- Validação de entradas textuais

---

### ▶️ **SubstituindoLinks**
Identificação e substituição de URLs em texto.
- Uso de Regex para detectar links em uma string
- Substituição automática por `[LINK]`
- Manipulação de padrões complexos em texto
